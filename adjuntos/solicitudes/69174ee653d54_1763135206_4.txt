sistema Motus

Es un sistema de gestión de créditos, llevando la gestión y la comunicación con los bancos
desde la solicitud por parte del cliente hasta su probación, el sistematiene una integración con pipedray 
para improtar los led que genera el usuario vendedor 

funciona de la siguiente manera 

el usuario vendedor crea el led en pipedray, luego el usuario administrador o comercial importa el led y empieza la gestión
con el banco, el usuario comercial o administrador también completa la imformacion, el banco aprueba o declina la propuesta   

Existen 3 tipos de usuarios 

administrador
comercial
banco

el administrador cuenta con un dashboard general, donde puede ver el resumen de todas las solicitudes y sus estados actuales 
el usuario administrador puede:
	- crear, editar o eliminar Usuarios del sistema, de todos los tipos, comercial, admin, banco 
	- crear, editar, eliminar e importar led de las Solicitudes de credito
	- crear, editar o eliminar bancos y usuarios del banco

el usuario comercial tiene un dashboard que solo muestra la información asignada a el 
	- crear, editar, eliminar e importar led de las Solicitudes de crédito
	- puede intercambiar información con el usuario Banco a cerca de la solicitud asignada 

el usuario banco tiene un dashboard que solo muestra la información asignada a el 
	- solo puede ver las Solicitudes de crédito asignadas a el
	- puede aceptar o declinar las solicitudes 
	- puede intercambiar información con el usuario comercial a cerca de la solicitud asignada
